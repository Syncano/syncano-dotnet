SyncServer Class
Class makes possible to connect to Syncano Sync Server over Tcp. Like in class Syncano, in Rest Api module, you can manage your Syncano Instance using it, but also it provides api for subscribing and listening for real time notifications that inform you, what is happening with your Syncano Instance right now.
Remarks
The Sync Server is meant to facilitate notification/subscription processing. To control what you subscribe to or what you want to send, refer to the notification and subscription API documentation. Once you are subscribed to and connected to the Sync Server, you will start receiving real-time notifications.
